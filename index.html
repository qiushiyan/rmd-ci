线性模型

用简单线性模型探究地区对销售量对影响，公式为:

\[ 销售量 = \beta_o + \beta_1地区A + \beta_1地区B + \cdots + \beta_1地区Z \]

mod <- lm(sales ~ region, data = sales_dat)

region_coefs <- mod$coefficients[-1]
max_idx <- which.max(region_coefs)
